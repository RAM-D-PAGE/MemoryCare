อัลบั้มความทรงจำส่วนตัว

ผู้ดูแลหรือญาติอัปโหลดรูปภาพ บุคคล สถานที่สำคัญ เพื่อให้ผู้ป่วยดูและเล่าเรื่องประกอบ กระตุ้น Autobiographical Memory

  • จัดหมวดหมู่ตามช่วงชีวิต (วัยเด็ก, วัยรุ่น, ผู้ใหญ่)
  • บันทึกเสียงเล่าเรื่องประกอบแต่ละรูป
  • พยาบาลเห็นความถี่ในการใช้งาน
🎵

ดนตรีบำบัดความทรงจำ

Playlist เพลงตามยุคสมัย ใช้หลัก Reminiscence Bump เปิดเพลงยุคที่ผู้ป่วยอายุ 15–25 ปี กระตุ้นความจำระยะยาวได้ดีที่สุด

  • เลือกเพลงตามปีเกิดอัตโนมัติ
  • วัด Emotional Response หลังฟัง
  • บันทึกเรื่องราวที่นึกถึงจากเพลง
🧩

เกมฝึกสมองแบบปรับระดับ

เกมที่ออกแบบเฉพาะสำหรับฝึกความจำระยะสั้น ความสนใจ และการรับรู้ ปรับความยากอัตโนมัติตามผลการเล่น

  • จับคู่รูปบุคคลในครอบครัว
  • เรียงลำดับเหตุการณ์ในชีวิต
  • ทายชื่อสิ่งของในอดีต
📖

เรื่องเล่าชีวิต (Life Review)

พาผู้ป่วยทบทวนชีวิตอย่างมีโครงสร้าง ผ่านคำถามชวนคิดในแต่ละช่วงชีวิต บันทึกเป็น "หนังสือชีวิต" ส่วนตัว

  • 6 บท ครอบคลุมทุกช่วงชีวิต
  • บันทึกเป็นเสียงหรือข้อความ
  • แชร์ให้ญาติอ่านได้
📊

Dashboard พยาบาล

พยาบาลติดตามผู้ป่วยหลายรายพร้อมกัน เห็นความถี่การใช้งาน คะแนนเกม และสัญญาณเตือนเมื่อพฤติกรรมเปลี่ยน

  • กราฟแสดงพัฒนาการรายสัปดาห์
  • แจ้งเตือนเมื่อผู้ป่วยขาดกิจกรรม
  • Export รายงานเป็น PDF

🧠 ด้านความจำและสติปัญญา

  • MMSE (Mini-Mental State Examination)
  • MoCA (Montreal Cognitive Assessment)
  • คะแนนเกมในแอป (In-app Score)
  • จำนวนความทรงจำที่ระลึกได้

💛 ด้านอารมณ์และจิตใจ

  • GDS (Geriatric Depression Scale)
  • DASS-21 (ความเครียด/วิตกกังวล)
  • Self-Esteem Scale (Rosenberg)
  • แบบวัดคุณภาพชีวิต SF-36

📱 ด้านการใช้งานแอป

  • ความถี่และระยะเวลาการใช้งาน
  • ความสม่ำเสมอ (Adherence Rate)
  • แบบวัดความพึงพอใจผู้ใช้ (SUS)
  • ความพึงพอใจของญาติ/พยาบาล
